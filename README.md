# AIverse

Aiverse는 AI로 생성한 이미지, 영상, 음악 등 디지털 창작물을 등록하고, 사용자가 크레딧으로 구매할 수 있는 AI 디지털 콘텐츠 마켓입니다.

## 문제 정의

최근 생성형 AI 기술의 발전으로 개인도 이미지, 영상, 음악과 같은 디지털 창작물을 빠르게 제작할 수 있게 되었습니다. 하지만 AI로 만든 창작물을 체계적으로 등록하고, 판매하고, 구매할 수 있는 구조는 아직 제한적입니다.

창작자 입장에서는 AI로 만든 결과물을 단순히 개인 작업물로 보관하는 데 그치지 않고, 다른 사용자가 활용할 수 있는 디지털 자산으로 판매하고 싶은 수요가 있습니다. 그러나 창작물의 유형, 생성 도구, 사용 라이선스, 가격, 판매 내역을 함께 관리하기 어렵다는 문제가 있습니다.

구매자 입장에서는 직접 AI 도구를 다루지 않아도 필요한 이미지, 영상, 음악 등을 빠르게 찾고 구매하고 싶은 수요가 있습니다. 하지만 AI 창작물은 일반 콘텐츠와 달리 어떤 AI 도구로 생성되었는지, 상업적으로 사용할 수 있는지, 구매 후 어떤 권한이 제공되는지 확인하기 어렵습니다.

따라서 AI 창작물을 등록, 탐색, 구매, 보관할 수 있고, 크레딧 기반 결제를 통해 구매 이력과 콘텐츠 접근 권한을 관리할 수 있는 서비스가 필요합니다.

## 서비스 기획

Aiverse는 AI 창작자와 구매자를 연결하는 AI 디지털 콘텐츠 마켓입니다. 창작자는 AI로 생성한 이미지, 영상, 음악 등을 업로드하고 가격과 라이선스를 설정할 수 있습니다. 구매자는 카테고리와 태그를 기준으로 창작물을 탐색하고, 크레딧을 충전한 뒤 원하는 콘텐츠를 구매할 수 있습니다.

실제 결제 연동은 목업 형태로 구현하되, 서비스 구조상 결제 성공 이후 크레딧이 충전되고, 콘텐츠 구매 시 크레딧이 차감되며, 구매한 콘텐츠에 대한 접근 권한이 생성되는 흐름을 중심으로 설계합니다.

## 주요 사용자

### 창작자

AI 도구를 활용해 이미지, 영상, 음악 등의 콘텐츠를 제작하고 판매하려는 사용자입니다.

창작자는 콘텐츠를 등록할 때 제목, 설명, 카테고리, 태그, 가격, 생성 도구, 라이선스 정보를 설정할 수 있습니다. 또한 등록한 콘텐츠의 판매 내역과 수익 현황을 확인할 수 있습니다.

### 구매자

AI 창작물을 검색하고 구매해 활용하려는 사용자입니다.

구매자는 콘텐츠 목록에서 원하는 창작물을 확인하고, 미리보기와 라이선스 정보를 확인한 뒤 크레딧으로 구매할 수 있습니다. 구매한 콘텐츠는 보관함에서 다시 확인하거나 다운로드할 수 있습니다.

## 핵심 기능

1. AI 창작물 등록

창작자는 AI로 생성한 이미지, 영상, 음악을 등록할 수 있습니다.
콘텐츠 등록 시 제목, 설명, 유형, 카테고리, 태그, 가격, 생성 도구, 라이선스 정보를 입력합니다.

2. AI 창작물 탐색

사용자는 등록된 AI 창작물을 최신순, 인기순, 가격순으로 확인할 수 있습니다.
이미지, 영상, 음악 등 콘텐츠 유형별로 탐색할 수 있고, 태그를 통해 원하는 주제의 창작물을 찾을 수 있습니다.

3. 크레딧 충전

사용자는 콘텐츠 구매에 사용할 크레딧을 충전할 수 있습니다.
실제 결제 연동은 목업으로 처리하지만, 결제 성공 시 사용자의 크레딧 잔액이 증가하는 구조로 설계합니다.

4. 콘텐츠 구매

사용자는 보유한 크레딧으로 원하는 AI 창작물을 구매할 수 있습니다.
구매가 완료되면 크레딧이 차감되고, 해당 콘텐츠에 대한 구매 이력과 접근 권한이 생성됩니다.

5. 구매 보관함

사용자는 구매한 콘텐츠를 보관함에서 확인할 수 있습니다.
보관함에서는 구매한 콘텐츠의 미리보기, 라이선스 정보, 구매일, 다운로드 기능을 제공합니다.

6. 창작자 판매 대시보드

창작자는 자신이 등록한 콘텐츠 수, 판매 횟수, 누적 판매 크레딧을 확인할 수 있습니다.
이를 통해 어떤 콘텐츠가 많이 판매되었는지 확인하고, 이후 창작 방향을 정할 수 있습니다.

## 서비스 흐름

사용자가 회원가입 또는 로그인을 합니다.
창작자는 AI 창작물을 등록합니다.
구매자는 콘텐츠 목록에서 원하는 창작물을 탐색합니다.
구매자는 크레딧 상품을 선택하고 충전합니다.
구매자는 보유 크레딧으로 콘텐츠를 구매합니다.
구매 완료 시 크레딧이 차감되고 구매 권한이 생성됩니다.
사용자는 보관함에서 구매한 콘텐츠를 확인하거나 다운로드합니다.
창작자는 판매 대시보드에서 판매 현황을 확인합니다.

## 기대 효과

Aiverse를 통해 AI 창작자는 자신의 결과물을 판매 가능한 디지털 자산으로 관리할 수 있습니다. 구매자는 직접 AI 도구를 다루지 않아도 필요한 창작물을 빠르게 탐색하고 구매할 수 있습니다.

또한 결제, 크레딧, 구매 권한, 다운로드 이력을 분리하여 설계함으로써 단순 콘텐츠 전시 서비스가 아니라 실제 거래 흐름을 가진 AI 창작물 마켓 구조를 표현할 수 있습니다.

## Getting Started

### Frontend

```bash
cd frontend
yarn install
yarn dev
```

앱은 **http://localhost:5173** 에서 확인할 수 있습니다.

### Backend

```bash
cd backend
./gradlew bootRun        # Linux / macOS
gradlew.bat bootRun      # Windows
```

API 서버는 **http://localhost:8080** 에서 실행됩니다.
